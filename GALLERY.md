# Shader Gallery

A collection of examples built with the ESHI framework. All examples run in real-time on both CPU and GPU backends.

| Preview | Name | Description |
| :--- | :--- | :--- |
| <img src="resources/warp.gif" width="300"> | **Warp** | A domain warping noise effect commonly used in clouds and terrain generation. Demonstrates `sampler2D` usage in CUDA. |
| <img src="resources/deepsea.gif" width="300"> | **Deep Sea** | Volumetric lighting and raymarching simulating underwater caustic patterns. |
| <img src="resources/fractal.gif" width="300"> | **Fractal** | Iterative Mandelbulb/Julia set rendering showing high-precision floating point capabilities. |
| <img src="resources/bubbles.gif" width="300"> | **Bubbles** | Particle system simulation demonstrating efficient distance field collisions. |
| <img src="resources/neon.gif" width="300"> | **Neon** | Glow effects and post-processing bloom implementation. |
| <img src="resources/raymarch.gif" width="300"> | **Raymarch** | Basic raymarching setup with soft shadows and ambient occlusion. |
| <img src="resources/voronoi.gif" width="300"> | **Voronoi** | Cellular noise patterns used for organic textures. |
| <img src="resources/starfield.gif" width="300"> | **Starfield** | 3D projection of points simulating high-speed space travel. |
